# LabyrinthGame
# Подземный Лабиринт

Текстовый квест. Вы в подземелье, нужно найти выход.

## Запуск

### Windows

Скачайте `labyrinth-windows.exe` и запустите его.

Если появилось окно «Windows защитила ваш компьютер» — нажмите **Подробнее**, затем **Выполнить в любом случае**.

Если ничего не происходит при двойном клике — попробуйте через командную строку:
```
cd путь\до\папки
labyrinth.exe
```

Если вместо русского текста отображаются кракозябры:
```
chcp 65001
labyrinth.exe
```

---

### macOS

Скачайте файл `labyrinth-macos`, откройте Терминал и напишите:

```bash
chmod +x labyrinth
./labyrinth
```

Если macOS не даёт запустить («разработчик неизвестен»):

```bash
xattr -d com.apple.quarantine labyrinth
./labyrinth
```

Или через **Системные настройки → Конфиденциальность и безопасность** и нажать «Разрешить».

## Управление

```
w a s d  — движение
e        — взаимодействие (сундук, дверь, NPC рядом)
i        — инвентарь
m        — сохранить
0        — выйти
```

Подойдите вплотную к объекту и нажмите `e`.

## Перекомпиляция

Если бинарник по какой-то причине не запускается — можно собрать из исходников.

Сначала скачайте весь репозиторий — нужны папки `src/`, `include/`, а также `Makefile` и `CMakeLists.txt`. Проще всего нажать **Code → Download ZIP** на странице репозитория и распаковать.

Нужен любой компилятор с поддержкой **C++17**.

**Через Make:**
```bash
cd labyrinth
make clean && make
```

**Через CMake:**
```bash
cd labyrinth
cmake -B build
cmake --build build
```

CMake сам найдёт установленный компилятор, подходит для любой платформы. На Windows работает с MinGW, MSVC и Clang. На macOS компилятор можно установить командой `xcode-select --install`.
